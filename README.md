# @things-factory/node-casscale

Nowsys NMD client module for nodejs.

## Install

```bash
$ npm install @things-factory/node-nowsys-nmd --save
```

## Build

```bash
yarn install 
yarn build
```

## Examples

### run test server

```bash
cd test-server
python nmd-erver.py
```

### run the example

```bash
cd examples
yarn install
node simple.js
```
