# @things-factory/node-nowsys-nmd

Nowsys NMD client module for nodejs.
(Nowsys Metal Detector)

## Build

```bash
yarn install 
yarn build
```

## Examples

### run test server

```bash
cd test-server
python nmd-erver.py
```

### run the example

```bash
cd examples
yarn install
node simple.js
```
